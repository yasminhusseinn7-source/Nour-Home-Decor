# Nour Home Decor 🏺

A single-page frontend website for a fictional home décor & lifestyle shop, built with pure HTML, CSS, and JavaScript. Created as a freelance training project simulating a real client brief.

## About

Nour Home Decor is an Arabic (RTL) website for a home décor brand selling handcrafted candles, ceramics, and home accessories. The project was built to practice working with an Arabic-first design, RTL layout, and a warm, natural color palette distinct from other portfolio projects.

The site presents a curated selection of products rather than a full catalog, encouraging visitors to reach out via WhatsApp or Instagram for the complete collection — matching how the (fictional) client currently sells through social media.

## Features

- 🏠 Single-page layout: Home, Collection, About, and Contact sections
- 🕯️ Featured product cards (candles, vases, baskets, trays, and more)
- 🌿 Warm, earthy color palette (sand beige + olive green accent)
- 🔤 Fully Arabic content with right-to-left (RTL) layout
- 📱 Fully responsive design across desktop, tablet, and mobile
- 📞 Direct contact via WhatsApp and Instagram (no cart/checkout, by design)

## Tech Stack

- HTML5 (`lang="ar"`, `dir="rtl"`)
- CSS3 (Flexbox, Grid, media queries)
- Vanilla JavaScript
- Google Fonts: Cormorant Garamond, Nunito

## Design

| Element | Value |
|---|---|
| Background | `#f5f0e8` |
| Primary text / headings | `#3e3a34` |
| Accent (buttons, hover, prices) | `#7d8c6a` |
| Secondary text | `#9c9284` |
